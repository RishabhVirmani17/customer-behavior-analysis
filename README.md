# Customer Behavior Analysis Dashboard

## 📌 Overview

This project analyzes customer shopping behavior using Python, PostgreSQL, and Power BI to generate business insights.

## 🛠️ Tech Stack

* Python (Pandas)
* PostgreSQL
* SQL
* Power BI

## 📊 Key Insights

* Clothing category generates highest revenue
* Subscription users spend more on average
* Seasonal trends impact sales performance
* Gender distribution shows customer segmentation
